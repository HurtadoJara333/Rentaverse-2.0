# Panoramas

Coloca tu foto 360° equirectangular aquí con el nombre:

  showroom.jpg

## Requisitos de la foto
- Formato: JPG o PNG
- Tipo: Equirectangular (2:1 aspect ratio — e.g. 4096×2048 px)
- Cómo obtenerla:
  - Cámara 360° (Ricoh Theta, Insta360, GoPro Max)
  - Google Street View app (modo 360°)
  - iPhone con modo Panorama (requiere conversión a equirectangular)
  - Foto esférica de Google Maps

## Para desarrollo/pruebas
Descarga una foto de prueba gratuita de:
  https://polyhaven.com/hdris  (descargar en formato HDR o EXR, convertir a JPG)
  https://www.flickr.com/groups/equirectangular/  (buscar "parking" o "garage")

## Tamaño recomendado
- Desarrollo: 2048×1024 px (~500 KB)
- Producción: 4096×2048 px (~2 MB)
- Ultra: 8192×4096 px (~8 MB, para pantallas de alta resolución)
